# 그레이엄 가치투자 워치리스트

## Problem Statement
HMW — 매번 여러 사이트를 오가며 재무제표를 뒤지는 대신, 관심 종목을 넣기만 하면 벤저민 그레이엄 기준으로 "이 종목이 가치투자에 부합하는지"를 한눈에 보여줄 수 있을까?
(틀 변경: "전체 시장에서 추천받기" → "내가 지정한 종목을 빠르게 판정받기". 3시간 제약에서 전체 스캔은 불가능하므로, 추천의 방향을 "발굴"에서 "판정"으로 바꿈)

## Recommended Direction
사용자가 종목명을 검색해서(한 글자만 입력해도 코스피200 마스터 리스트 기반 자동완성 제공) 워치리스트에 추가하면, 공공데이터(또는 DART) API에서 시세와 재무 데이터를 가져와 그레이엄의 방어적 투자자 기준(PER, PBR, 부채비율, 순자산 등 4~5개 지표) 대비 몇 개를 만족하는지 체크리스트/스코어로 보여주는 개인용 대시보드.

전체 상장사를 자동으로 훑어 추천 리스트를 뽑는 버전은 이상적이지만, 3시간 안에 API 데이터 커버리지 확인 + 배치 스캔 로직까지 만들기엔 리스크가 큽니다. "검색해서 추가 → 판정" 흐름이 훨씬 작고, 같은 핵심 가정(재무 데이터를 API로 가져올 수 있다)을 검증하면서도 실제로 완성 가능성이 높습니다. 종목명 자동완성은 코스피200으로 범위를 좁혀 정적 마스터 리스트로 처리하므로 별도의 실시간 API 리스크 없이 구현할 수 있습니다.

## Key Assumptions to Validate
- [ ] **(가장 중요, 제일 먼저 확인)** 공공데이터포털 또는 DART API에서 종목코드 기준으로 PER/PBR/부채비율/순자산 같은 재무 데이터를 무료로 조회할 수 있다 — API 문서/샘플 응답을 실제로 호출해서 5분 내로 확인
- [ ] 코스피200 종목의 코드+이름 마스터 리스트를 정적 파일로 미리 확보할 수 있다 — Should Be True (안 되면 최악의 경우 소수 종목만 하드코딩해서 데모 가능)
- [ ] 그레이엄 기준을 4~5개로 단순화해도 "괜찮아 보이는 종목"이라는 판단에 실질적으로 도움이 된다고 스스로 느낀다
- [ ] 종목 하나를 검색해서 판정받는 흐름이 3시간 안에 끝까지(검색→조회→판정 표시) 동작한다

## MVP Scope
**포함**:
- 종목명 한 글자 이상 입력 시 코스피200 마스터 리스트 기반 부분 일치 자동완성
- 종목 검색/추가, 워치리스트 저장(로컬)
- 종목별 그레이엄 체크리스트(4~5개 지표 + 만족 개수)
- 최신 시세 표시

**제외**:
- 전체 시장(코스피200 밖) 자동 스캔
- 그래프/과거 시세
- 실시간 갱신
- 여러 그레이엄 변형 기준 지원

## Not Doing (and Why)
- 전체 상장사 자동 스캔·랭킹 — 3시간 제약에서 배치 처리·API 레이트리밋 리스크가 너무 큼. 검증되면 다음 확장으로
- 코스피200 밖 종목 지원 — 마스터 리스트 크기·검색 속도를 3시간 안에 다루기 쉬운 수준으로 제한
- 실제 매수/증권사 연동 — 개인 투자용 리서치 도구일 뿐, 주문 실행까지 갈 필요 없음
- 여러 투자 전략(그레이엄 외 피터 린치 등) 지원 — 학습 목적상 하나의 기준을 제대로 만드는 게 우선
- 공개 서비스화(투자 조언 문구 등 법적 이슈) — 본인 전용 도구로 범위 한정

## Open Questions
- 공공데이터포털/DART API 키 발급이 이미 되어 있는지, 아니면 지금 신청부터 해야 하는지 (신청 승인 대기시간이 있으면 3시간 안에 위험)
- 코스피200 마스터 리스트(코드+이름)를 어디서 받을지 — 공공데이터포털 CSV, KRX 정보데이타시스템, 또는 다른 정적 소스
